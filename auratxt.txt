Write an Ansible playbook to install and configure a web server.
Step 1: Install WSL (Ubuntu) on Windows
wsl --install


Step 2: Install Ansible inside your new Linux environment
sudo apt update
1. Install Ansible: 
sudo apt install ansible -y


2. Check version
ansible --version


Step 3: Create a workspace
   1. Get to supper user mode
sudo su 
mkdir devops_exam
cd devops_exam


Step 4: Create the Inventory file
   1. Type nano inventory.ini to open a simple text editor.
   2. Paste the following line:
localhost ansible_connection=local


   3. Save and exit by pressing Ctrl + X, then Y, then Enter.


Step 5: Write the Playbook
   1. Type nano webserver.yml
   2. Carefully paste the following code:


---
- name: Install and configure a web server
  hosts: localhost
  become: yes 
  tasks:
    - name: Update apt cache
      apt:
        update_cache: yes


    - name: Install Apache web server
      apt:
        name: apache2
        state: present


    - name: Ensure Apache is running and starts on boot
      service:
        name: apache2
        state: started
        enabled: yes


    - name: Create a custom index.html page
      copy:
        dest: /var/www/html/index.html
        content: |
          <h1>Welcome to my Ansible Web Server!</h1>
          <p>My DevOps practical exam was a success.</p>


Step 6: Run the Playbook
ansible-playbook -i inventory.ini webserver.yml --ask-become-pass


Step 7: Check the ip of the wsl then enter the ip in the browser
Ip a




Write a Dockerfile to containerize a Python/Node.js application and run it.
Step 1: Install WSL (Ubuntu) on Windows
wsl --install


Phase 1: Preparation (Installing Docker)
sudo apt update
sudo apt install docker.io -y


Phase 2: Creating the Files
Step 1: Create a new workspace
mkdir docker_exam
cd docker_exam


Step 2: Create the Python file
Type nano app.py
Type exactly what you need for your exam:
print("Hello Billal sir")






Step 3: Create the Dockerfile
   1. Type nano Dockerfile
   2. Paste these exact 4 lines:
FROM python:3.9-slim
WORKDIR /app
COPY app.py .
CMD ["python", "app.py"]
   3. Save and exit (Ctrl + X, then Y, then Enter).


Step 4: Build the Docker Image
Run this command (don't forget the period . at the very end, it tells Docker to look in your current folder!):
sudo docker build -t my-exam-app .


Step 5: Run the Container
sudo docker run my-exam-app

Practical 9 - Deploying a Simple Application Using Kubernetes(Windows)

 Open Docker Desktop Settings
         * Find the whale icon 🐳 in your system tray (near the clock)
         * Right-click it and select Settings
         * Or open Docker Desktop and click the gear icon (⚙️) in the top-right corner 
1.2 Enable Kubernetes
         * Click on Kubernetes in the left sidebar 
         * Check the box that says "Enable Kubernetes" 
         * Click "Apply & Restart" at the bottom
1.3 Wait for Kubernetes to Start

         * You'll see the status in the Docker Dashboard footer
         * When ready, the whale icon may show a Kubernetes indicator
